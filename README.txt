Citizen AI clean repo-ready core file set

Replace these files in your repo:
- requirements.txt
- citizen_ai_bot/services.py
- citizen_ai_bot/static_data.py
- citizen_ai_bot/formatters.py
- citizen_ai_bot/cogs/starcitizen.py

Then restart locally with:
python -m citizen_ai_bot

This pack assumes your existing project already has:
- models.py
- uex_client.py
- config.py
- utils.py
- bot bootstrap / __main__.py
